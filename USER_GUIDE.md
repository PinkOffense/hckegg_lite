## ðŸ“ Managing Your Hens

### Adding a New Hen

1. Tap **"Add Hen"** from the main menu
2. Fill in the details:
   - **Identification** - Unique ID or name (e.g., "H001", "Brownie")
   - **Breed** - Select from common breeds (Rhode Island Red, Leghorn, etc.)
   - **Date of Birth** - When the hen was hatched
   - **Batch** - Assign to a group (optional)
   - **Status** - Active, Sold, Deceased

3. Tap **"Save"**

### Viewing Hen Details

- Tap any hen from the list to see:
  - Production history
  - Health records
  - Age and productivity stats
  - Cost analysis

### Organizing by Batches

**Batches** help you group hens acquired at the same time:

1. Go to **"Batches"** menu
2. Tap **"Create New Batch"**
3. Enter:
   - Batch name (e.g., "Spring 2025")
   - Start date
   - Initial quantity
   - Notes

4. Assign hens to batches when adding them

## ðŸ¥š Recording Daily Production

### How to Log Eggs

1. Tap **"Daily Production"** on the dashboard
2. Enter today's date (auto-filled)
3. Record eggs by:
   - **Total Eggs** - Quick entry for all eggs
   - **By Batch** - Track production per group
   - **By Quality** - Sort by size/grade

### Quality Grades
- **XL** - Extra Large (>73g)
- **L** - Large (63-73g)
- **M** - Medium (53-63g)
- **S** - Small (<53g)
- **Broken/Dirty** - Non-sellable

### Tips
- Record production at the same time daily
- Note any unusual patterns
- Check the weekly chart for trends

## ðŸ†˜ Troubleshooting

### App Won't Sync
- Check internet connection
- Verify account is active
- Try manual sync
- Contact support if issue persists

### Data Not Showing
- Pull down to refresh
- Check date range filters
- Verify hens are marked "Active"
- Clear cache and restart

### Can't Add Production
- Ensure date is valid (not future)
- Check that you have active hens
- Verify batch exists if using batch entry

### Export Not Working
- Check storage permissions
- Ensure enough device storage
- Try a smaller date range
- Update app to latest version
